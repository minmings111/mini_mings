**# Mini Mings**

**>** JWT 기반 사용자 인증 시스템

**## 📌 프로젝트 소개**

Spring Boot와 JWT를 활용한 안전한 사용자 인증 시스템입니다.

**## 🏗 프로젝트 구조**

**-\*\***backend/\*\* - Spring Boot 백엔드

** -** JWT 인증 시스템

** -** Spring Security 설정

** -** MyBatis + MySQL

**-\*\***frontend/\*\* - 프론트엔드 (개발 예정)

**## ✨ 주요 기능**

**### ✅ 구현 완료**

**- JWT 토큰 기반 인증 시스템**

**- Spring Security 통합**

**- 데이터베이스 스키마 설계 (소셜 로그인 대비)**

**- 역할 기반 접근 제어 (RBAC)**

**### 🚧 개발 예정**

**- 회원가입/로그인 API 구현**

**- 웹 프론트엔드 연동**

**- OAuth 2.0 소셜 로그인 (Google, Kakao)**

**- 사용자 프로필 관리**

**## 🛠 기술 스택**

\***\*Backend\*\***

**- Java 17**

**- Spring Boot 3.x**

**- Spring Security + JWT**

**- MyBatis**

**- MySQL 8.0**

\***\*Frontend** (예정)\*\*

**- React / Next.js**

**- TypeScript**
